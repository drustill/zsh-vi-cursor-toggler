# zsh-vi-cursor

**zsh helper to toggle cursor shape when switching between vi-modes**

## Install

```bash
# With oh my zsh
git clone https://github.com/drustill/zsh-vi-cursor.git {$ZSH_CUSTOM:-$ZSH/custom}/plugins/zsh-vi-cursor.git

# In ~/.zshrc
plugins+=(zsh-vi-cursor)

# Reload zsh
exec zsh -l
```
